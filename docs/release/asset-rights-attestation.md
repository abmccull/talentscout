# TalentScout asset-rights attestation

Attestation date: July 13, 2026

The TalentScout release owner states that all shipped image and audio assets covered by `docs/asset-provenance.json` are their own sourced, project-authored, or AI-generated assets and that they own the commercial rights required to use and distribute them with TalentScout.

This declaration covers the current hashed inventory under:

- `public/images/avatars/`
- `public/images/backgrounds/`
- `public/images/branding/`
- `public/images/steam/`
- `public/audio/ambience/`
- `public/audio/music/`

The inventory is regenerated by `npm run audit:asset-provenance`. Any new or replaced asset must be assigned to a provenance group and receive source/rights review before the release gate is rerun.

Evidence class: release-owner commercial-rights attestation. This records the owner's declaration and is not an independent legal opinion.
